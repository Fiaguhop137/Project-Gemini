# Project Gemini

A Minecraft experiment. Two players who will never meet, sharing a single world that neither will ever keep.

## Rules
- Each player occupies the world for exactly one in game day(20 minutes). 
    - This can be checked with the day_counter feature in the 1.21.9+ f3 screen. 
- At the end of that cycle, the world save must be zipped, exported, and passed to the next player.
- Players may never communicate outside of the world data itself.
- No voice calls, no chat logs, and no texts.
- The only proof of the other player's existence is the physical evidence left behind in the blocks.
- Do not place signs or write books.

## Importing Project Gemini
1. Close Minecraft and all related processes to prevent corruption.
2. Open your worlds folder.
   - For the Modrinth app, this is typically profiles/[modpack name]/saves.
3. Download the attached .zip file into that folder.
4. Extract the archive using your preferred unzip tool or the built-in Windows extraction feature.
5. Launch Minecraft and play the world.

## Exporting Project Gemini
1. Close Minecraft and all related processes to prevent corruption. 
2. Go to your worlds folder and open the Project Gemini world folder.
3. Select everything inside.
4. Compress it into a .zip file named "Project Gemini - Day x".
5. Upload the archive to the shared repository.
    - Delete the world from your saves folder after uploading.
